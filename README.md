# Sync Company
> ### 그룹웨어 프로젝트
 #### 그룹웨어 프로그램을 참고하여 기획하고 여러 기능을 추가하여
 #### 원활한 의사소통과 정보 공유와 신속한 의사 결정이 이루어질 수 있는 업무 환경을 제공하며
 
#### 편리하게 이용할 수 있는 그룹웨어 시스템을 만들자는 목표로 프로젝트를 기획하였습니다.
